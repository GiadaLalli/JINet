    Explore

    Self-contained, self-sustainable, client-based computation platform for clinical and multi-view data analysis in the healthcare context. JINet hosts numerous applications that you can run in the safety of your web browser without your data ever leaving your own machine. Applications are drawn from a wide range of sources including peer-reviewed pipelines, while citing original code.

    Use programming languages you're familiar with to write applications. Python, R, Julia, and Javascript are supported runtimes.

    This project implements a flexible and ever-evolving solution interoperability. JINet supports researchers' efforts by making software easily accessible through a simple point-and-click interface.

    JINet is created by researchers for researchers, aimed at expediting advancements in science and shedding light on interesting yet not fully recognized methods. If you wish to support JINet, please consider submitting your wrapper through the dedicated contribute section or by visiting our public git repository.

    Multi-view data

    Datasets that contain information from multiple sources or perspectives, often presenting a comprehensive picture of a complex system or phenomenon. In biomedical research, multi-view data encompasses various types of information, including but not limited to multi-omics, multi-modal, and imaging data.

    Clinical data

    Information collected during clinical care, medical research, or healthcare administration. It encompasses a wide range of data types related to patients' health status, medical history, diagnostic tests, treatments, and outcomes. Clinical data can be collected through various sources, including electronic health records (EHRs), medical imaging, laboratory tests, patient-reported outcomes, and administrative databases.

    Application

    A function written in a supported language to perform a computation based on some user inputs. Applications only run on clients machines and data is never transferred to our server.

    Self-contained

    A self-contained application is a software framework that includes all the necessary components and dependencies required to run independently on a target system without relying on external resources. In other words, it is a standalone application that can be executed on a computer or device without the need for additional installations or configurations.

    Self-sustainable

    A self-sustainable application is a software system capable of operating independently and autonomously without relying heavily on external resources or constant human intervention. Such applications are designed to efficiently manage their own resources, adapt to changing conditions, and continue functioning over an extended period without significant external support.
