    Explore

    Self-contained, self-sustainable, client-based computation platform for clinical and multi-view data analysis in the healthcare context. JINet hosts numerous applications that you can run in the safety of your web browser without your data ever leaving your own machine. Applications are drawn from a wide range of sources including peer-reviewed pipelines, while citing original code.

    Use programming languages you're familiar with to write applications. Python, R, Julia, and Javascript are supported runtimes.

    This project implements a flexible and ever-evolving solution for interoperability. JINet supports researchers' efforts by making software easily accessible through a simple point-and-click interface.

    JINet is created by researchers for researchers, aimed at expediting advancements in science and shedding light on interesting yet not fully recognized methods. If you wish to support JINet, please consider submitting your wrapper through the dedicated contribute section or by visiting our public git repository.

    Multi-view data

    Multi-view data is like putting together pieces of a puzzle from different angles. In biomedical research, it's a collection of various types of information, such as genetics, imaging, and more, providing a full picture of complex biological systems.

    Clinical data

    Clinical data is like a comprehensive health file, covering everything from basic patient info like age and gender to detailed records of medical history, tests, treatments, and outcomes. It's gathered from electronic health records, lab tests, doctor's notes, and other sources, helping healthcare professionals make informed decisions.

    Application

    A function written in a supported programming language to perform a computation based on some user inputs. Applications only run on clients machines and data is never transferred to our server.

    Self-contained

    A self-contained application is like a self-sufficient toolbox that has everything it needs inside. It's ready to run on its own, without requiring any additional tools or resources from outside. Just open it up, and it's ready to go!

    Self-sustainable

    A self-sustainable application is like a smart assistant that can take care of itself. It knows how to manage its resources efficiently, adapt to changes, and keep running smoothly without needing constant attention or assistance. It's like having a reliable helper who can handle tasks independently.

    Interoperability

    The ability of two or more systems or components to exchange information and to use the information that has been exchanged (from IEEE ICD, 1991).
